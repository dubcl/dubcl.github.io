Which Way Is Up

Posted on July 6, 2007
![](http://hectigo.net/puskutraktori/whichwayisup/pictures/head.png) [Home](http://hectigo.net/puskutraktori/whichwayisup/) Which Way Is Up es un juego en 2D creado en python, aun no lo e probado bien pero se ve entretenido, me recuerda a [lode runner](http://en.wikipedia.org/wiki/Lode_Runner) pero en este hay que rotar la pantalla activando unos switch y encontrando unas llaves (por lo visto). Tambien hay ke recalcar ke se encuentra bajo la licencia GPL 2.0 y tambien tiene contenido [Creative Commons 3.0](http://creativecommons.org/licenses/by/3.0/) y esta disponible para windows y Linux bueno si lo kieren probar lo pueden bajarlo desde su [pagina web](http://hectigo.net/puskutraktori/whichwayisup/) Newest version **last updated 2007-05-23**: - [Beta 0.7.0 for Windows](http://hectigo.net/puskutraktori/whichwayisup/whichwayisup_b070_bin.zip), 3600 KB - [Beta 0.7.0, Python source code](http://hectigo.net/puskutraktori/whichwayisup/whichwayisup_b070.zip), 730 KB
